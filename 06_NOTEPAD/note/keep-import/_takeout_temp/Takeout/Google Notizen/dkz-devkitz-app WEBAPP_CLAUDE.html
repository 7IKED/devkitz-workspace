## 2️⃣ MODUL-KOMBINATIONEN

### **Content erstellen & speichern:**
```
User Input
→ content.js (Template laden, Blöcke füllen)
→ content.js (saveContent() aufrufen)
→ storage.js (DBStorage.add/update)
→ ui.js (Dashboard Update triggern)
→ core.js (Notification zeigen)
```

### **Projekt exportieren:**
```
User klickt Export
→ export.js (exportMarkdown/PDF/etc)
→ export.js (logExport())
→ storage.js (DBStorage.add zu 'exports')
→ Datei wird als Download bereitgestellt
```

### **Drive Sync:**
```
User klickt Sync
→ sync.js (Auth prüfen)
→ sync.js (performSync())
→ storage.js (driveQueue holen)
→ [extern] Google Drive API Upload
→ storage.js (Queue leeren, lastSync setzen)
→ core.js (Stats update)
```

### **Präferenzen ändern:**
```
User ändert Einstellung
→ ui.js (Form Update)
→ core.js (Auto-Save nach 2s)
→ storage.js (LocalStorage.set)
→ ui.js (applyTheme() für visuelle Updates)
```

### **Backup erstellen:**
```
User klickt Backup
→ core.js (createBackup())
→ storage.js (exportAll())
→ JSON-Blob erstellen
→ Browser-Download triggern
```
