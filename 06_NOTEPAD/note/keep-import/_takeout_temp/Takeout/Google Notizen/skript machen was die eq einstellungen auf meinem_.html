batchKopierenBearbeiten@echo off
setlocal EnableDelayedExpansion

:: Tool-Check
set tools=EqualizerAPO Realtek Audio Console Dolby Atmos
set found=0

:check_tools
for %%t in (%tools%) do (
reg query "HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall" /s /f "%%t" >nul 2>&1
if !errorlevel! == 0 (
echo Tool gefunden: %%t
set found=1
set chosenTool=%%t
goto tool_found
)
)

echo Kein Equalizer-Tool gefunden.
echo Vorschlag: Installiere Equalizer APO für erweiterte Audioeinstellungen.
pause
exit

:tool_found
echo Gewähltes Tool: %chosenTool%
if /i "%chosenTool%" == "EqualizerAPO" (
echo Equalizer APO wird verwendet.
:: Hier EQ-Einstellungen ändern
) else if /i "%chosenTool%" == "Realtek Audio Console" (
echo Realtek Audio Console wird verwendet.
:: Realtek-Einstellungen ändern
) else (
echo Dolby Atmos erkannt - keine Automatisierung möglich.
)
pause

FxSound bietet keine direkte Schnittstelle zur externen Steuerung von EQ-Werten über Skripte. Dennoch gibt es einen praktikablen Weg: die Erstellung und Verwendung benutzerdefinierter Presets.

🔧 Vorgehensweise: Benutzerdefiniertes Preset erstellen und automatisieren


FxSound öffnen und EQ einstellen:


Starte FxSound.


Stelle die Equalizer-Werte gemäß deinen Wünschen ein:


129 Hz: +2 dB


238 Hz: +1 dB


444 Hz: 0 dB


805 Hz: 0 dB


1.36 kHz: 0 dB


2.52 kHz: 0 dB


4.40 kHz: +1 dB


7.93 kHz: +2 dB


12.57 kHz: +3 dB




Passe die Effekteinstellungen entsprechend an:


Klarheit: 75%


Ambiente: 25%


Surround-Klang: 50%


Dynamikverstärkung: 50%


Bass-Verbesserung: 75%






Preset speichern:


Klicke auf das Diskettensymbol oder wähle "Preset speichern".


Gib einen eindeutigen Namen ein, z. B. "MeinEQ".




Preset-Datei sichern:


Navigiere zu %AppData%\FxSound\Presets.


Suche die Datei mit dem Namen deines Presets, z. B. MeinEQ.fx.


Kopiere diese Datei an einen sicheren Ort zur Sicherung.




Preset automatisch laden (optional):


FxSound lädt standardmäßig das zuletzt verwendete Preset beim Start. Stelle sicher, dass dein gewünschtes Preset zuletzt aktiv war.


Alternativ kannst du die Preset-Datei in den Ordner %AppData%\FxSound\Presets kopieren und überschreiben, um sicherzustellen, dass deine Einstellungen geladen werden.





⚠️ Wichtige Hinweise


Keine direkte Skriptsteuerung: FxSound unterstützt derzeit keine Befehlszeilenparameter zum direkten Laden von Presets oder Einstellen von EQ-Werten.


Manuelle Anpassungen erforderlich: Änderungen an den EQ- oder Effekteinstellungen müssen manuell über die Benutzeroberfläche vorgenommen werden.


Preset-Dateien sind spezifisch: Die .fx-Dateien sind spezifisch für FxSound und sollten nicht mit anderen Programmen bearbeitet werden.



🛠️ Alternative: Equalizer APO
Wenn du eine detailliertere Kontrolle über deine Audioeinstellungen wünschst, könnte Equalizer APO eine geeignete Alternative sein. Es bietet:


Skriptbasierte Konfiguration: Einstellungen können über Textdateien angepasst werden.


Breite Kompatibilität: Funktioniert mit den meisten Windows-Audioausgabegeräten.


Erweiterbarkeit: Unterstützt Plugins wie Peace GUI für eine benutzerfreundlichere Oberfläche.



📌 Fazit
Während FxSound keine direkte Skriptsteuerung für EQ-Einstellungen bietet, kannst du durch das Erstellen und Verwenden benutzerdefinierter Presets deine gewünschten Einstellungen speichern und bei Bedarf laden. Für fortgeschrittene Anforderungen empfiehlt sich der Einsatz von Equalizer APO.

🔧 Equalizer APO: Konfigurierbare Schnittstelle
Equalizer APO verwendet einfache Textdateien zur Konfiguration, die im Installationsverzeichnis unter C:\Program Files\EqualizerAPO\config\ gespeichert sind. Diese Dateien enthalten Befehle wie Preamp, Filter und GraphicEQ, die es dir ermöglichen, präzise Audioanpassungen vorzunehmen.

🛠️ Beispiel: Equalizer-Einstellungen
Hier ist ein Beispiel für eine Konfigurationsdatei, die deinen Anforderungen entspricht:
plaintextKopierenBearbeitenPreamp: 0 dB
GraphicEQ: 10 -84; 129 2; 238 1; 444 0; 805 0; 1360 0; 2520 0; 4400 1; 7930 2; 12570 3

Diese Einstellungen passen die Verstärkung bei verschiedenen Frequenzen an, um den gewünschten Klang zu erzielen.

🚀 Automatisierung mit Batch-Dateien
Du kannst Batch-Dateien verwenden, um verschiedene Presets schnell zu laden. Hier ist ein Beispiel:
batchKopierenBearbeiten@echo off
copy "C:\EQ-Presets\MeinEQ.txt" "C:\Program Files\EqualizerAPO\config\config.txt" /Y
echo Preset 'MeinEQ' angewendet!
exit

Speichere diesen Code in einer .bat-Datei und führe sie aus, um das gewünschte Preset zu laden.
