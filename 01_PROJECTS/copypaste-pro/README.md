## Features

- **Clipboard History** ‚Äî Stores up to 200 entries with localStorage persistence
- **1-Click Copy** ‚Äî Copy any entry back to clipboard via `navigator.clipboard.writeText`
- **Favorites** ‚Äî Pin/Unpin important snippets for quick access
- **Fulltext Search** ‚Äî Live filter across all clipboard entries
- **Auto-Detect Type** ‚Äî Automatically tags entries as URL, Email, Code, or Text with colored badges
- **Category Filters** ‚Äî Filter by All, URLs, Code, Emails, Text, or Favorites
- **Keyboard Shortcut** ‚Äî `Ctrl+V` to paste with snackbar notification
- **Timestamps** ‚Äî Relative timestamps (e.g., "2m ago", "3h ago") for each entry
- **Bulk Operations** ‚Äî Select multiple entries for bulk deletion
- **Clear All** ‚Äî Confirm dialog before clearing entire history
- **Export JSON** ‚Äî Download full clipboard history as formatted JSON
- **Responsive Design** ‚Äî Card layout on mobile, optimized list view on desktop
- **i18n** ‚Äî English (default) and German language support
- **High Contrast Mode** ‚Äî Enhanced neon glow for accessibility
- **Glassmorphism UI** ‚Äî Premium dark theme with neon accents
