# QR Launcher — DEVKiTZ™

> QR-Code Generator und Launcher fuer schnellen Modul-Zugriff

## Features

- QR-Code Generierung aus URLs und Text
- Modul-Schnellstart via QR-Code Scan
- Batch QR-Erstellung fuer mehrere Module
- QR-Code Download als PNG/SVG
- Kurzlink-Integration

## Tech Stack

- Vanilla HTML5 + CSS3 + JavaScript ES6+
- DkZ Design System v2 (Glassmorphism)
- localStorage (Offline-First)

## Starten

Datei `index.html` im Browser oeffnen oder ueber das DkZ Dashboard Hub aufrufen.

## Pfad

`01_PROJECTS/01_dashboard/modules/qr-launcher/`
