# DkZ Drive Hub — DEVKiTZ™

> 00-99 File Intelligence mit Auto-Sortierung, Duplikat-Scan und Research Index

## Features

- Ordner-Struktur Visualisierung (00-99 System)
- Auto-Sortieren Funktion
- Duplikat-Erkennung mit 5-Punkt-Pruefung
- Research Index Verwaltung
- Synchronisations-Log und Historie

## Tech Stack

- Vanilla HTML5 + CSS3 + JavaScript ES6+
- DkZ Design System v2 (Glassmorphism)
- localStorage (Offline-First)

## Starten

Datei `index.html` im Browser oeffnen oder ueber das DkZ Dashboard Hub aufrufen.
