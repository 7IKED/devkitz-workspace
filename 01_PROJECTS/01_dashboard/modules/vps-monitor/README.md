# VPS Monitor — DEVKiTZ™

> Server-Monitoring fuer Hostinger VPS Instanzen

## Features

- CPU/RAM/Disk Monitoring in Echtzeit
- Service Health-Checks (HTTP, SSH, Docker)
- Uptime-Tracking und Verfuegbarkeits-Historie
- Alert-Benachrichtigungen bei Ausfaellen
- Multi-Server Uebersicht

## Tech Stack

- Vanilla HTML5 + CSS3 + JavaScript ES6+
- DkZ Design System v2 (Glassmorphism)
- localStorage (Offline-First)

## Starten

Datei `index.html` im Browser oeffnen oder ueber das DkZ Dashboard Hub aufrufen.

## Pfad

`01_PROJECTS/01_dashboard/modules/vps-monitor/`
