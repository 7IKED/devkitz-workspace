# NanoBot Center — DEVKiTZ™

> NanoBot Agent-System Verwaltung und Konfiguration

## Features

- Bot-Fleet Uebersicht aller NanoBots
- Agent Konfiguration und Parametersteuerung
- Custom Theme Builder fuer Bot-Interfaces
- Shortcut-Verwaltung und Tastenkuerzel
- Onboarding-Wizard fuer neue Bots

## Tech Stack

- Vanilla HTML5 + CSS3 + JavaScript ES6+
- DkZ Design System v2 (Glassmorphism)
- localStorage (Offline-First)

## Starten

Datei `index.html` im Browser oeffnen oder ueber das DkZ Dashboard Hub aufrufen.
