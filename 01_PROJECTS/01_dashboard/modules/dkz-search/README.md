# DkZ Search — DEVKiTZ™

> Intelligente Suche ueber alle Module mit Prompt Generator und Builder Chain

## Features

- Volltextsuche ueber alle DkZ Module
- Prompt Generator fuer KI-Anfragen
- Builder Chain Schnellzugriff
- CSS Variables Referenz
- Ralph Loop Navigation
- Fuzzy-Matching und Ergebnis-Ranking

## Tech Stack

- Vanilla HTML5 + CSS3 + JavaScript ES6+
- DkZ Design System v2 (Glassmorphism)
- localStorage (Offline-First)

## Starten

Datei `index.html` im Browser oeffnen oder ueber das DkZ Dashboard Hub aufrufen.

## Pfad

`01_PROJECTS/01_dashboard/modules/dkz-search/`
