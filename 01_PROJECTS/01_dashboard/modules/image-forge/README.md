# ImageForge Studio — DEVKiTZ™

> Bild-Generierung und -Bearbeitung Studio

## Features

- KI-Bildgenerierung mit Prompt-Editor
- Galerie-Verwaltung fuer generierte Bilder
- Batch-Generierung mehrerer Varianten
- Bild-Parameter Konfiguration
- Export und Download

## Tech Stack

- Vanilla HTML5 + CSS3 + JavaScript ES6+
- DkZ Design System v2 (Glassmorphism)
- localStorage (Offline-First)

## Starten

Datei `index.html` im Browser oeffnen oder ueber das DkZ Dashboard Hub aufrufen.

## Pfad

`01_PROJECTS/01_dashboard/modules/image-forge/`
