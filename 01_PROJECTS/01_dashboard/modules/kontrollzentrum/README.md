# Kontrollzentrum — DEVKiTZ™

> Zentrales Steuerungspanel fuer alle DEVKiTZ Systeme

## Features

- System-Uebersicht aller Module und Dienste
- Quick Actions Panel fuer haeufige Operationen
- Modul-Schnellzugriff und Navigation
- Status-Ampel System (Gruen/Gelb/Rot)
- Echtzeit Health-Monitoring

## Tech Stack

- Vanilla HTML5 + CSS3 + JavaScript ES6+
- DkZ Design System v2 (Glassmorphism)
- localStorage (Offline-First)

## Starten

Datei `index.html` im Browser oeffnen oder ueber das DkZ Dashboard Hub aufrufen.

## Pfad

`01_PROJECTS/01_dashboard/modules/kontrollzentrum/`
