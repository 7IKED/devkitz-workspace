# DkZ Copilot — Analyse, Tutorial, Blaupause — DEVKiTZ™

> Interaktive Copilot-Praesentation mit System-Architektur, Builder-Tutorial und Ampel-System

## Features

- System-Architektur Visualisierung
- 3-Tab-Panel (AI, Builder, Tools) Dokumentation
- Builder Schritt-fuer-Schritt Tutorial
- Ampel-System fuer Provider-Eignung
- Cost Viewer mit Preisvergleich

## Tech Stack

- Vanilla HTML5 + CSS3 + JavaScript ES6+
- DkZ Design System v2 (Glassmorphism)
- localStorage (Offline-First)

## Starten

Datei `index.html` im Browser oeffnen oder ueber das DkZ Dashboard Hub aufrufen.
