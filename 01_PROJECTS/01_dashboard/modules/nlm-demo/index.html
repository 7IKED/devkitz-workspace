Ralph-Loop™

Jeder Task bekommt frischen Kontext — kein Context Drift. 6 Phasen, reproduzierbar, validiert.

1. LESEN Input
PRD, Constitution, Agents laden
2. SPAWN Init
Neue Instanz, frischer Kontext
3. EXECUTE Code
Developer™ implementiert
4. VERIFY QA
Reviewer™ + Tester™ pruefen
5. COMMIT Save
Git commit + Status update
6. LOOP Repeat
