# AppScript Builder — DEVKiTZ™

> Google Apps Script Generator mit Template-Bibliothek und Deploy-Integration

## Features

- Google Apps Script Code-Generierung
- Template-Bibliothek fuer gaengige Scripts
- WissenHub und NEXUZ Integration
- Iceberg Archiv-Anbindung
- Batch-Export aller Scripts

## Tech Stack

- Vanilla HTML5 + CSS3 + JavaScript ES6+
- DkZ Design System v2 (Glassmorphism)
- localStorage (Offline-First)

## Starten

Datei `index.html` im Browser oeffnen oder ueber das DkZ Dashboard Hub aufrufen.
