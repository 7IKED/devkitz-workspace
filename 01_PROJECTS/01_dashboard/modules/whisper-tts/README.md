# Whisper TTS — Audio Studio — DEVKiTZ™

> Text-to-Speech und Audio-Verarbeitung mit Whisper Integration

## Features

- Whisper TTS Integration fuer Sprachsynthese
- Audio-Aufnahme direkt im Browser
- Transkription von Audio zu Text
- Multi-Stimmen Auswahl und Konfiguration
- Audio-Datei Export und Verwaltung

## Tech Stack

- Vanilla HTML5 + CSS3 + JavaScript ES6+
- DkZ Design System v2 (Glassmorphism)
- Web Audio API
- localStorage (Offline-First)

## Starten

Datei `index.html` im Browser oeffnen oder ueber das DkZ Dashboard Hub aufrufen.

## Pfad

`01_PROJECTS/01_dashboard/modules/whisper-tts/`
