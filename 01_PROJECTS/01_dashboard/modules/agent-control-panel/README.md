# Agent Control Panel — DEVKiTZ™

> AI Agent Steuerung, Ueberwachung und J.A.R.V.I.S. Integration

## Features

- BMAD Agenten-Status Monitoring und Steuerung
- J.A.R.V.I.S. KI-Assistenz Integration
- Ralph-Loop Task-Zuweisung und Kontrolle
- Agent-Konfiguration speichern und laden
- Dashboard-Ansicht mit Echtzeit-Status

## Tech Stack

- Vanilla HTML5 + CSS3 + JavaScript ES6+
- DkZ Design System v2 (Glassmorphism)
- localStorage (Offline-First)

## Starten

Datei `index.html` im Browser oeffnen oder ueber das DkZ Dashboard Hub aufrufen.
