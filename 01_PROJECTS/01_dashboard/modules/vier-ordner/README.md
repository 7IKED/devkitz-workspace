# 4-Ordner-System — DEVKiTZ™

> TODO, MEETING, IDEA und USE CASE Ordner-Management nach Produktivitaets-Methode

## Features

- 4-Zonen Ordner-System (TODO, MEETING, IDEA, USE CASE)
- TODO Verwaltung mit Priorisierung
- Meeting-Notes Organisation und Archiv
- Ideen-Tracking mit Tags
- Use-Case Dokumentation und Verwaltung

## Tech Stack

- Vanilla HTML5 + CSS3 + JavaScript ES6+
- DkZ Design System v2 (Glassmorphism)
- localStorage (Offline-First)

## Starten

Datei `index.html` im Browser oeffnen oder ueber das DkZ Dashboard Hub aufrufen.
