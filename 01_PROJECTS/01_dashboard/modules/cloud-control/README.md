# Cloud Control — DEVKiTZ™

> Multi-Cloud Steuerung mit Drive, Cloudflare, Oracle und mehr

## Features

- Multi-Cloud Provider Uebersicht (Drive, Cloudflare, Oracle)
- Cloud-Service Health-Checks
- User-Verwaltung und Zugriffssteuerung
- Hub-Navigation zu allen Cloud-Diensten
- Quick Actions fuer haeufige Operationen

## Tech Stack

- Vanilla HTML5 + CSS3 + JavaScript ES6+
- DkZ Design System v2 (Glassmorphism)
- localStorage (Offline-First)

## Starten

Datei `index.html` im Browser oeffnen oder ueber das DkZ Dashboard Hub aufrufen.

## Pfad

`01_PROJECTS/01_dashboard/modules/cloud-control/`
