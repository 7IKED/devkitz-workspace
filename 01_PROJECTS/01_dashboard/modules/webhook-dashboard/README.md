# Webhook Dashboard — DEVKiTZ™

> Webhook-Verwaltung und Event-Monitoring

## Features

- Webhook-Endpoint Verwaltung und Konfiguration
- Event-Log Anzeige mit Zeitstempel
- Payload-Inspektion und Debug-Ansicht
- Test-Trigger Funktion fuer manuelle Tests
- Webhook-Historie und Statistiken

## Tech Stack

- Vanilla HTML5 + CSS3 + JavaScript ES6+
- DkZ Design System v2 (Glassmorphism)
- localStorage (Offline-First)

## Starten

Datei `index.html` im Browser oeffnen oder ueber das DkZ Dashboard Hub aufrufen.
