# DkZ Cost Command Center — DEVKiTZ™

> Kosten-Dashboard mit Token-Rechner, Projekt-Builder und Provider-Vergleich

## Features

- Token-Rechner fuer LLM API Kosten
- Projekt-Builder mit Kostenvoranschlag
- Provider-Vergleich (OpenAI, Anthropic, Google, etc.)
- Kosten-Simulator fuer Szenarien
- Hub-Navigation zu allen Diensten

## Tech Stack

- Vanilla HTML5 + CSS3 + JavaScript ES6+
- DkZ Design System v2 (Glassmorphism)
- localStorage (Offline-First)

## Starten

Datei `index.html` im Browser oeffnen oder ueber das DkZ Dashboard Hub aufrufen.

## Pfad

`01_PROJECTS/01_dashboard/modules/cost-dashboard/`
