# Kanban Board — NEXUZ[DkZ] — DEVKiTZ™

> Kanban-Board fuer Projektmanagement und Task-Tracking

## Features

- Drag-and-Drop Karten-Verwaltung
- Spalten-Management (TODO, IN PROGRESS, DONE, etc.)
- Task-Priorisierung und Labels
- localStorage Persistenz
- NEXUZ-OS Integration

## Tech Stack

- Vanilla HTML5 + CSS3 + JavaScript ES6+
- DkZ Design System v2 (Glassmorphism)
- localStorage (Offline-First)

## Starten

Datei `index.html` im Browser oeffnen oder ueber das DkZ Dashboard Hub aufrufen.
