# Blog Commander — DEVKiTZ™

> Blogger API Automatisierung und Multi-Blog Management

## Features

- Google Blogger API Integration mit OAuth
- Multi-Blog Post-Erstellung und Veroeffentlichung
- Page-Management und Verwaltung
- Blog-uebergreifende Volltext-Suche
- Aktivitaets-Log und Post-Vorschau

## Tech Stack

- Vanilla HTML5 + CSS3 + JavaScript ES6+
- DkZ Design System v2 (Glassmorphism)
- localStorage (Offline-First)
- Google Blogger API v3

## Starten

Datei `index.html` im Browser oeffnen oder ueber das DkZ Dashboard Hub aufrufen.

## Pfad

`01_PROJECTS/01_dashboard/modules/blog-commander/`
