# Design Gallery — DEVKiTZ™

> UI Pattern Galerie mit Components, Layouts, Patterns und Effects

## Features

- UI Pattern Sammlung mit Kategorien (Components, Layouts, Patterns, Effects)
- Glassmorphism Design-Beispiele
- Live Preview und Code-Ansicht
- Responsive Card-Layout
- Filter und Suche

## Tech Stack

- Vanilla HTML5 + CSS3 + JavaScript ES6+
- DkZ Design System v2 (Glassmorphism)
- localStorage (Offline-First)

## Starten

Datei `index.html` im Browser oeffnen oder ueber das DkZ Dashboard Hub aufrufen.

## Pfad

`01_PROJECTS/01_dashboard/modules/design-gallery/`
