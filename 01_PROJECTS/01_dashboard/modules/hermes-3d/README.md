# Hermes 3D — Agent Ecosystem — DEVKiTZ™

> 3D-Visualisierung des gesamten Agent-Oekosystems

## Features

- 3D Agent-Netzwerk Darstellung
- Interaktive Kamera-Steuerung (Rotation, Zoom)
- Echtzeit Agent-Status Anzeige
- Verbindungs-Visualisierung zwischen Agenten
- Oekosystem-Uebersicht mit Details-on-Hover

## Tech Stack

- Vanilla HTML5 + CSS3 + JavaScript ES6+
- DkZ Design System v2 (Glassmorphism)
- localStorage (Offline-First)
- WebGL / Canvas 3D

## Starten

Datei `index.html` im Browser oeffnen oder ueber das DkZ Dashboard Hub aufrufen.

## Pfad

`01_PROJECTS/01_dashboard/modules/hermes-3d/`
