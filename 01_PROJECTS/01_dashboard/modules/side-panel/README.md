# DkZ Side Panel — DEVKiTZ™

> AI Search Side Panel fuer kontextbezogene Hilfe

## Features

- Side-Panel Integration in alle Module
- KI-gestuetzte Suche mit Kontext-Erkennung
- Kontextbezogene Ergebnisse basierend auf aktuellem Modul
- Quick Actions fuer haeufige Operationen
- Minimierbares Panel-Layout

## Tech Stack

- Vanilla HTML5 + CSS3 + JavaScript ES6+
- DkZ Design System v2 (Glassmorphism)
- localStorage (Offline-First)

## Starten

Datei `index.html` im Browser oeffnen oder ueber das DkZ Dashboard Hub aufrufen.
