# Graphify — Playbook Viewer — DEVKiTZ™

> Visuelle Graph-Darstellung von Playbooks und Workflows

## Features

- Graph-Visualisierung von Playbooks
- Interaktive Knoten-Navigation
- Zoom und Pan Steuerung
- Playbook Parsing und Rendering
- Verknuepfungs-Darstellung

## Tech Stack

- Vanilla HTML5 + CSS3 + JavaScript ES6+
- DkZ Design System v2 (Glassmorphism)
- localStorage (Offline-First)

## Starten

Datei `index.html` im Browser oeffnen oder ueber das DkZ Dashboard Hub aufrufen.

## Pfad

`01_PROJECTS/01_dashboard/modules/graphify/`
