# Blog Gallery — DEVKiTZ™

> Content Hub fuer das Blog-Netzwerk mit Kategorie-Filtern

## Features

- Blog-Uebersicht aller Verticals und Kanaele
- Kategorie-Filter (Tech, Tutorial, Research, News)
- Content-Vorschau und Schnellzugriff
- Responsive Card-Layout
- Statistik-Dashboard

## Tech Stack

- Vanilla HTML5 + CSS3 + JavaScript ES6+
- DkZ Design System v2 (Glassmorphism)
- localStorage (Offline-First)

## Starten

Datei `index.html` im Browser oeffnen oder ueber das DkZ Dashboard Hub aufrufen.

## Pfad

`01_PROJECTS/01_dashboard/modules/blog-gallery/`
