# NEXUZ WissenHub — DEVKiTZ™

> Knowledge System und zentrale Wissens-Datenbank des Oekosystems

## Features

- Wissens-Archiv mit Volltextsuche
- Such- und Filter-System (nach Tags, Typ, Datum)
- Tag-basierte Organisation aller Artefakte
- Artefakt-Dreifach-Verankerung (Iceberg + Hub + Copilot)
- Research-Archive Integration

## Tech Stack

- Vanilla HTML5 + CSS3 + JavaScript ES6+
- DkZ Design System v2 (Glassmorphism)
- localStorage (Offline-First)
- Apache Iceberg / DuckDB

## Starten

Datei `index.html` im Browser oeffnen oder ueber das DkZ Dashboard Hub aufrufen.

## Pfad

`01_PROJECTS/01_dashboard/modules/wissen-hub/`
