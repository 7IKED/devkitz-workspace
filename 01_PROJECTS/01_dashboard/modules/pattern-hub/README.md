# Pattern Hub — DEVKiTZ™

> Templates, Skills und Feedback Sammlung

## Features

- Template-Bibliothek fuer Modul-Entwicklung
- Skill-Katalog mit Uebersicht und Suche
- Feedback-System fuer Verbesserungsvorschlaege
- Pattern-Suche und Filter
- Quick-Copy fuer Templates

## Tech Stack

- Vanilla HTML5 + CSS3 + JavaScript ES6+
- DkZ Design System v2 (Glassmorphism)
- localStorage (Offline-First)

## Starten

Datei `index.html` im Browser oeffnen oder ueber das DkZ Dashboard Hub aufrufen.
