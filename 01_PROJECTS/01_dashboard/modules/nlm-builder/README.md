# NLM Builder — DEVKiTZ™

> NotebookLM Batch Content-Generierung und Quellen-Management

## Features

- NotebookLM API Integration
- Batch Content-Erstellung fuer alle Content-Typen
- Podcast-Generierung aus Quellen
- Quellen-Management und Upload
- Content-Vorschau und Export

## Tech Stack

- Vanilla HTML5 + CSS3 + JavaScript ES6+
- DkZ Design System v2 (Glassmorphism)
- localStorage (Offline-First)
- NotebookLM API

## Starten

Datei `index.html` im Browser oeffnen oder ueber das DkZ Dashboard Hub aufrufen.
