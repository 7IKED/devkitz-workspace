# DEEPKEEP Tresor — DEVKiTZ™

> Sicherer Tresor fuer sensible Daten, Credentials und geschuetzte Dateien

## Features

- Verschluesselter Datenspeicher mit Tresor-Funktion
- Desktop/Downloads automatische Pruefung
- Volltext-Suche in gesicherten Dateien
- CSV Export fuer Datenuebersicht
- Auto-Lock Timer fuer Sicherheit

## Tech Stack

- Vanilla HTML5 + CSS3 + JavaScript ES6+
- DkZ Design System v2 (Glassmorphism)
- localStorage (Offline-First)

## Starten

Datei `index.html` im Browser oeffnen oder ueber das DkZ Dashboard Hub aufrufen.
