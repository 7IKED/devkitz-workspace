# LLM Arena — DEVKiTZ™

> Model Benchmark und Vergleichs-Tool fuer LLMs

## Features

- LLM Model-Vergleich (OpenAI, Anthropic, Google, etc.)
- Benchmark-Tests mit konfigurierbaren Prompts
- Antwort-Qualitaets Bewertung und Scoring
- Kosten-Leistungs Analyse pro Provider
- Ergebnis-Export und Historie

## Tech Stack

- Vanilla HTML5 + CSS3 + JavaScript ES6+
- DkZ Design System v2 (Glassmorphism)
- localStorage (Offline-First)

## Starten

Datei `index.html` im Browser oeffnen oder ueber das DkZ Dashboard Hub aufrufen.

## Pfad

`01_PROJECTS/01_dashboard/modules/llm-arena/`
