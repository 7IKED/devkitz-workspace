# CodeRabbit + Live-Check — DEVKiTZ™

> Code-Review Dashboard mit CodeRabbit, OpenSpec und Copilot Integration

## Features

- CodeRabbit API Integration fuer automatische Code-Reviews
- Live-Check Status-Monitor
- OpenSpec Validierung
- Copilot Integration
- Rules-Editor fuer Review-Regeln

## Tech Stack

- Vanilla HTML5 + CSS3 + JavaScript ES6+
- DkZ Design System v2 (Glassmorphism)
- localStorage (Offline-First)
