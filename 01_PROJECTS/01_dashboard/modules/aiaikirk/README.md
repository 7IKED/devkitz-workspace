# AiAiKirk Copilot — DEVKiTZ™

> KI-Copilot Chat-Interface mit Preview, Export und Kontext-Hilfe

## Features

- Chat-Interface fuer kontextbezogene KI-Assistenz
- Live Preview-Ansicht fuer generierte Inhalte
- Chat-Export Funktionalitaet
- Einklappbares Side-Panel
- Modul-Navigation und Wissens-Abfragen

## Tech Stack

- Vanilla HTML5 + CSS3 + JavaScript ES6+
- DkZ Design System v2 (Glassmorphism)
- localStorage (Offline-First)

## Starten

Datei `index.html` im Browser oeffnen oder ueber das DkZ Dashboard Hub aufrufen.
