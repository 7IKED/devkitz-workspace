# DkZ Design Studio v3 — DEVKiTZ™

> Design System Editor mit Look Like, Be Unique und Import/Export Workflow

## Features

- CSS Variable Live-Editor
- Theme Builder mit Vorschau
- Design Token Management
- Import/Export von Design-Systemen
- Look Like und Be Unique Modus

## Tech Stack

- Vanilla HTML5 + CSS3 + JavaScript ES6+
- DkZ Design System v2 (Glassmorphism)
- localStorage (Offline-First)

## Starten

Datei `index.html` im Browser oeffnen oder ueber das DkZ Dashboard Hub aufrufen.
