# CLOUDIA² Sortier-Engine — DEVKiTZ™

> Automatische Datei-Sortierung mit Provider, Queue, Regeln, Sync und Metadaten

## Features

- Provider-basierte Datei-Klassifikation
- Sortier-Queue mit Batch-Verarbeitung
- Regelbasierte Sortierung konfigurierbar
- Synchronisations-Engine fuer Cloud-Dienste
- Metadaten-Verwaltung und -Analyse

## Tech Stack

- Vanilla HTML5 + CSS3 + JavaScript ES6+
- DkZ Design System v2 (Glassmorphism)
- localStorage (Offline-First)

## Starten

Datei `index.html` im Browser oeffnen oder ueber das DkZ Dashboard Hub aufrufen.
