# Paperclip[DkZ] — DEVKiTZ™

> LLM Orchestrierung und Multi-Model Management

## Features

- Multi-LLM Steuerung (OpenAI, Anthropic, Google, OSS)
- Prompt-Routing an optimales Modell
- Model-Fallback Ketten konfigurierbar
- Token-Verbrauch Tracking pro Session
- Antwort-Vergleich zwischen Modellen

## Tech Stack

- Vanilla HTML5 + CSS3 + JavaScript ES6+
- DkZ Design System v2 (Glassmorphism)
- localStorage (Offline-First)

## Starten

Datei `index.html` im Browser oeffnen oder ueber das DkZ Dashboard Hub aufrufen.
