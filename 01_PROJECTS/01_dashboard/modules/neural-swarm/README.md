# Neural Swarm — DEVKiTZ™

> Neuronales Netzwerk Swarm-Visualisierung und -Steuerung

## Features

- Swarm-Intelligenz Visualisierung
- Agent-Netzwerk Graph-Darstellung
- Task Distribution ueber Agenten
- Echtzeit-Monitoring der Swarm-Aktivitaet
- Konfigurierbare Swarm-Parameter

## Tech Stack

- Vanilla HTML5 + CSS3 + JavaScript ES6+
- DkZ Design System v2 (Glassmorphism)
- Canvas / WebGL
- localStorage (Offline-First)

## Starten

Datei `index.html` im Browser oeffnen oder ueber das DkZ Dashboard Hub aufrufen.
