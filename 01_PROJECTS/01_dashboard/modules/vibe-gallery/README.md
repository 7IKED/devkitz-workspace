# Vibe Gallery — DEVKiTZ™

> Design Vibes Kollektion und Inspiration fuer UI-Entwicklung

## Features

- Vibe-basierte Design-Sammlung
- Stimmungs-Filter (Dark, Light, Neon, Minimal, etc.)
- Favoriten-System fuer bevorzugte Designs
- Design-Export als CSS/HTML Snippet
- Responsive Galerie-Ansicht

## Tech Stack

- Vanilla HTML5 + CSS3 + JavaScript ES6+
- DkZ Design System v2 (Glassmorphism)
- localStorage (Offline-First)

## Starten

Datei `index.html` im Browser oeffnen oder ueber das DkZ Dashboard Hub aufrufen.

## Pfad

`01_PROJECTS/01_dashboard/modules/vibe-gallery/`
