# Display und Performance Configurator — DEVKiTZ™

> 4K HDR Streaming und Display-Einstellungen Konfigurator

## Features

- 4K/HDR Display-Konfiguration
- Performance-Monitoring und Metriken
- Streaming-Optimierung fuer verschiedene Aufloesung
- Display-Profile speichern und laden
- Hardware-Erkennung

## Tech Stack

- Vanilla HTML5 + CSS3 + JavaScript ES6+
- DkZ Design System v2 (Glassmorphism)
- localStorage (Offline-First)

## Starten

Datei `index.html` im Browser oeffnen oder ueber das DkZ Dashboard Hub aufrufen.
