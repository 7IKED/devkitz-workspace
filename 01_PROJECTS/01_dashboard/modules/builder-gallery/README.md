# Builder Gallery — DEVKiTZ™

> Unified Asset Hub und Werkzeuge-Sammlung fuer alle DkZ Builder

## Features

- Zentrale Uebersicht aller Builder-Tools
- Asset-Verwaltung mit Vorschau
- Drag-and-Drop Upload
- Kategorie-basierte Sortierung und Navigation
- Quick-Access zu allen Builder-Modulen

## Tech Stack

- Vanilla HTML5 + CSS3 + JavaScript ES6+
- DkZ Design System v2 (Glassmorphism)
- localStorage (Offline-First)

## Starten

Datei `index.html` im Browser oeffnen oder ueber das DkZ Dashboard Hub aufrufen.

## Pfad

`01_PROJECTS/01_dashboard/modules/builder-gallery/`
