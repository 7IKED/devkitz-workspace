# BLACK8 (n8n) Builder — DEVKiTZ™

> ComfyUI Visual Workflow Builder fuer n8n Automationen

## Features

- Visueller Workflow-Editor im ComfyUI-Stil
- Verknuepfung mit Actions, Workflows und Agents Buildern
- TEN0R und BLACK8 Builder-Modus Umschaltung
- n8n Node-Bibliothek und Konfiguration
- Workflow Import/Export Funktionalitaet

## Tech Stack

- Vanilla HTML5 + CSS3 + JavaScript ES6+
- DkZ Design System v2 (Glassmorphism)
- localStorage (Offline-First)

## Starten

Datei `index.html` im Browser oeffnen oder ueber das DkZ Dashboard Hub aufrufen.
