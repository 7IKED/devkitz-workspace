# DkZ devkitz V2 - User Manual

## Introduction

DkZ devkitz is a local-first project management tool for developers and content creators.

## Features

- **Dual Editor**: Switch between Smart (Markdown) and Extended (Rich Text) modes.
- **Offline Storage**: All data is saved instantly to your browser.
- **Google Drive Sync**: Backup your projects to the cloud.

## Getting Started

1. Open `index.html`.
2. Go to **Settings** to configure your Name and Theme.
3. Click **New Project** to start writing.

## Keyboard Shortcuts

- `Ctrl + S`: Save Project
- `Ctrl + B`: Toggle Sidebar
- `Ctrl + E`: Export Project
